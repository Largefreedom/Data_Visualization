#### WordArtVideo

---

将一个人物视频转化为词云图视频

----

包含功能：

* download()
  * youget 下载视频
* download_danmu()
  * B站弹幕采集 — 
* Seg_img — 需借助于百度 API 
  * 人像分割；
* video_jpg
  * 将是视频切帧，转化为图像序列；
* Gen_wordart()
  * 生成词云图；
* image_video()
  * 图片序列合成视频；
